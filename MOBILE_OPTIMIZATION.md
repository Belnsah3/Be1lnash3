## 🎯 **Ключевые особенности:**

### **Адаптивная сетка:**
```css
/* Десктоп */
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));

/* Планшет */
grid-template-columns: repeat(2, 1fr);

/* Смартфон */
grid-template-columns: 1fr;

/* Ландшафт */
grid-template-columns: repeat(4, 1fr);
```

### **Breakpoints:**
- **768px** - планшеты и маленькие ноутбуки
- **480px** - смартфоны
- **Landscape** - горизонтальная ориентация

### **Touch-friendly:**
- Минимальная высота кнопок: **44px** (рекомендация Apple)
- Увеличенные отступы между элементами
- Плавный скролл для iOS
